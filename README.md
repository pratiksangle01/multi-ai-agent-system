# Multi AI Agent System 🤖🚀

A simple multi-agent AI system that simulates how different AI agents collaborate to complete tasks.

---

## 🔥 Features
- 🤖 Multiple AI agents working together
- 🔗 Sequential workflow execution
- 🧠 Simulates real-world AI systems
- ⚡ Fast and modular design

---

## 🧠 Agents Included

### 1. Research Agent
Analyzes the task and gathers initial information.

### 2. Reply Agent
Generates a response based on research data.

### 3. Strategy Agent
Suggests the next steps based on the generated reply.
